# App Hub Icon Brief

## Status

The current build uses the DHIS2 App Platform default/generated icon assets. A unique app icon should be created before App Hub submission.

## Recommended concept

Create a 512x512 PNG source icon using simple bold shapes, without the DHIS2 logo or copied public-health marks.

Suggested composition:

- Background: solid deep teal or blue-green field with high contrast.
- Foreground: a simplified shield or rounded diamond for decision support.
- Signal element: a small dog/animal silhouette or paw-like exposure marker, kept abstract and not cartoonish.
- Decision element: a clear check mark or branching path line inside the shield.
- Style: flat vector, thick shapes, readable at 48x48, no small text.

## Required exported assets

- 512x512 PNG master icon for App Hub listing.
- App Platform icon assets generated from the master icon during build/package preparation.
- Optional SVG source stored in design materials only if licensing is clear.

## Constraints

- Do not use the DHIS2 logo in the custom app icon.
- Do not use WHO, UKHSA, ministry, or government marks.
- Avoid medical-cross imagery unless local governance approves it.
- Verify contrast and recognizability at 48x48 and 192x192 sizes.
